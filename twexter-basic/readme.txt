===================
twexter 0.01 demo 
===================

about
-------------------
twexter 0.01 demos basic twexter functions
* chunk twext output
* xcroll input, including
* live preview

install
-------------------
unpack files in /twexter/ directory
* index.php
* jquery.js
* tabber.css
* tabber.js
* twext.neutral.php
* twext.php
* twext.xsl.php4
* twext.xsl.php5
* twext_tables.xsl
* twexter.css
* twexter.js
* xx.gif

place directory in php enabled url ie
http://yourwebsite.url/twexter

demo
-------------------
currently working w/ latest firefox only

beta test 
-------------------

twexter beta-testing in early june, including:
* saving .twx files

* file versioning system (save file history)
