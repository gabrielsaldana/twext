===================
Twexter 0.4 demo
===================

Features
-------------------
- A very clean and simple usable interface.
- Minimal memory and process footprint.
- No server side requirements, 100% javascript, CSS and HTML.
- All twexter functions are encapsulated as a single javascript file.

Install
-------------------
Unpack files in twexter/ directory

Place directory in your web server's public directory.
In Debian/Ubuntu is: /var/www/

Visit the project's url
http://yourwebserver.com/twexter

Requirements
-------------------
Currently working with Gecko engine based browsers (like Firefox) only.
No server side requirements, any webserver will work.
